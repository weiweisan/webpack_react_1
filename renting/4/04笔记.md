		最重要的就是 options的配置 https://echarts.baidu.com/option.html#title
		
	React:echarts-for-react https://github.com/hustcc/echarts-for-react
	Vue:vue-echarts
	
echarts-for-react 使用步骤
	1、安装两个包 yarn add echarts echarts-for-react -S
	2、导入 import ReactEcharts from 'echarts-for-react'
	3、使用 
		// render echarts option 
		<ReactEcharts option={this.getOption()} />
